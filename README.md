本程式為學生的學期成績列表，主要有三個功能：
1. 顯示所有學生的成績列表
2. 新增學生成績
3. 離開本程式
<br/>
而輸入完學生資料、平時成績、期中考成績、期末考成績後，本程式會自動算出最終的學期總成績，並顯示於列表上。
